Summary of What the Project Is
Domain: A digital showroom website for Toyota Rantauprapat (Auto2000 Rantauprapat, Labuhanbatu, North Sumatra, Indonesia) under PT. Astra International Tbk.
Core Technology Stack: Next.js 15 (App Router), React 19, TypeScript, Tailwind CSS, Framer Motion, and Radix UI.
Database & AI Integration:
Retrieval-Augmented Generation (RAG) search functionality utilizing a MySQL/TiDB vector database (mysql2).
Text embeddings calculated via a Hugging Face Space.
Recommendation content generated via Gemini AI API (@google/generative-ai).
